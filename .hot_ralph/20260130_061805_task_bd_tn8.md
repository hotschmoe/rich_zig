   - `syntax_highlighting: bool = true` - Enable/disable syntax highlighting
   - `syntax_theme: SyntaxTheme = SyntaxTheme.default` - Theme for code highlighting

3. **renderSourceContext()** - Enhanced to:
   - Detect language from file path using `Language.fromFilename()`
   - Use `highlightSourceLine()` when syntax highlighting is enabled
   - Blend highlight styles for target lines while preserving syntax colors

4. **highlightSourceLine()** - New method that:
   - Creates a `Syntax` instance for a single line of code
   - Applies the configured syntax theme
   - Returns highlighted segments

5. **blendHighlightStyle()** - New helper function that:
   - Preserves syntax colors (keywords, strings, etc.)
   - Adds bold/underline from the traceback highlight style
   - Applies background color from highlight when syntax has none
